# 90-Day Recovery Roadmap

## Phase 1: Stabilization (Weeks 1–4)

**Goal:** Reduce looping and anxiety. Establish baseline.

| Area | Guideline |
|------|-----------|
| Supplements | Magnesium + Alpha-GPC (core stack) |
| Social | Limit large gatherings. 1-on-1 only |
| Sleep | 10 hours minimum (first week) |
| Focus | Learn 1 thing per day (10 words, 1 name) |
| Exercise | Walking only — no high-intensity |
| Looping | Tactile grounding (stone or finger tap) |

## Phase 2: Rebuilding (Weeks 5–12)

**Goal:** Train hippocampus to store data again.

| Area | Guideline |
|------|-----------|
| Supplements | Add Creatine. Maintain DHA + Magnesium |
| Exercise | Light jogging, bodyweight |
| Social | 1 group gathering per week |
| Journaling | 3 mins/day — write 1 thing you learned |
| Memory | Spaced-repetition drills (Anki or paper cards) |

## Phase 3: Consolidation (Month 4+)

**Goal:** Return to normalcy.

| Area | Guideline |
|------|-----------|
| Supplements | Maintain DHA + Magnesium daily |
| Social | Regular social events |
| Focus | "Loop check" — when looping starts, touch nose 3 times |
| Professional | Consider neurofeedback or EMDR if symptoms persist |
